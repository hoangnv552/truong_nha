---Kiểm thử chức năng thêm sinh viên---

Yêu cầu:
	- Chạy trên nền ubuntu
	- Yêu cần cài thêm thư viện Selenium2Library
	- Chạy file ClassTest.txt

Danh sách các test:
	- Test trường hợp giáo viên có trùng tên
	- Test đúng các trường nhập vào
	- Test trường hợp tên nhập vào có chứa số ở đầu
	- Test trường hợp ngày sinh sai
	- Test trường hợp số điện thoại không đúng
